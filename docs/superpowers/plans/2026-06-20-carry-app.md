# Carry App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the Carry demo app — four screens (Chat A, Chat B, Dashboard, Access Control) on top of a server-enforced *gate-before-generation* policy and Answer Receipts, with one Walrus-anchored receipt.

**Architecture:** A single Next.js 16 app. A pure, unit-tested domain layer in `frontend/lib/` (policy gate, in-memory store, receipt builder) is invoked only from **server Route Handlers** under `app/api/*`, so the gate runs before any model call and blocked memory is never sent to the LLM. LLM and Walrus are accessed through swappable adapter interfaces with Mock implementations (zero external keys) selected by env, then swapped for OpenAI/Anthropic/Walrus. Client screens are thin UIs (built from the existing design-system primitives) that call the API.

**Tech Stack:** Next.js 16 (App Router) · React 19 · TypeScript (strict) · Tailwind v4 · Vitest (new) · OpenAI + Anthropic SDKs (real mode) · Walrus testnet HTTP (publisher/aggregator).

## Global Constraints

- **Gate before generation:** access is checked at retrieval, server-side; the model only ever receives allowed memory. Blocked namespaces are *never fetched*. `authorized` is always `true` for used memories; `blockedNamespaces` carries the real proof.
- **Mock-first:** every external dependency sits behind an interface with a Mock impl; `CARRY_MODE=mock` (default) runs the full flow with no keys. Real adapters activate via env (`OPENAI_API_KEY`, `ANTHROPIC_API_KEY`, `WALRUS_PUBLISHER`, `WALRUS_AGGREGATOR`).
- **Agents:** `agent-a` = writer (OpenAI/GPT-4o in real mode), `agent-b` = reader (Anthropic/Claude in real mode).
- **Namespaces (demo):** `diet`, `health`, `project`, `billing`.
- **Precise encryption language (no overclaiming):** "Revoke is enforced before retrieval in Carry's access layer today; Seal-based per-agent cryptographic enforcement is on the roadmap." Never claim live cryptographic revoke.
- **Next.js 16 caveat:** read `frontend/node_modules/next/dist/docs/` before using any Next API (route handlers, params are async, etc.). See `frontend/AGENTS.md`.
- **Code style:** no comments; small single-purpose files (split past ~180 lines); reuse design-system primitives (`Buttons`, `Eyebrow`, `Badge`, `Icon`, `cn`); recolor everything to Sui-blue `#4DA2FF`.
- **Git:** never add commit trailers; never commit `de.json`, `CLAUDE.md`, `PRD-*.md`.
- All commands run from `frontend/`.

### Task 2: Gate (recall — gate before generation)

**Files:**
- Create: `frontend/lib/gate.ts`
- Test: `frontend/lib/__tests__/gate.test.ts`

**Interfaces:**
- Consumes: `Memory`, `Policy`, `AgentId`, `NamespaceId`; `isAllowed`, `allowedNamespaces`.
- Produces: `type RecallResult = { memories: Memory[]; blockedNamespaces: NamespaceId[] }`, `recall(agent: AgentId, query: string, all: Memory[], policy: Policy): RecallResult`.

Semantics: a memory matches if its `content` shares a word with `query` (case-insensitive) OR the query is empty (recall-all). `memories` = matching memories in allowed namespaces only. `blockedNamespaces` = namespaces that (a) contain a matching memory and (b) the agent is denied — i.e., what the answer *could not* touch. Blocked memories are never returned.

```ts
import type { AgentId, Memory, NamespaceId, Policy } from "./types";
import { isAllowed } from "./policy";

export type RecallResult = { memories: Memory[]; blockedNamespaces: NamespaceId[] };

function matches(content: string, query: string): boolean {
  const q = query.trim().toLowerCase();
  if (!q) return true;
  const words = new Set(content.toLowerCase().split(/\W+/).filter(Boolean));
  return q.split(/\W+/).filter(Boolean).some((w) => words.has(w));
}

export function recall(agent: AgentId, query: string, all: Memory[], policy: Policy): RecallResult {
  const relevant = all.filter((m) => matches(m.content, query));
  const memories = relevant.filter((m) => isAllowed(agent, m.namespace, policy));
  const blockedNamespaces = Array.from(
    new Set(relevant.filter((m) => !isAllowed(agent, m.namespace, policy)).map((m) => m.namespace))
  );
  return { memories, blockedNamespaces };
}
```

```ts
export function buildReceipt(args: {
  agentId: AgentId;
  answerId: string;
  used: Memory[];
  blockedNamespaces: NamespaceId[];
  verifiedRefs: Set<string>;
  createdAt: string;

  return {
    answerId: args.answerId,
    agentId: args.agentId,
    usedMemories: args.used.map((m) => ({
      memoryId: m.memoryId,
      namespace: m.namespace,
      snippet: m.content,
      sourceAgent: m.sourceAgent,
      walrusRef: m.walrusRef,
      authorized: true,
      verified: args.verifiedRefs.has(m.walrusRef),
    })),
    blockedNamespaces: args.blockedNamespaces,
    createdAt: args.createdAt,
  };
}
```

```ts
export interface LLMProvider {

  complete(args: { agentId: AgentId; query: string; memories: Memory[] }): Promise<string>;

export class MockLLM implements LLMProvider {
```

```ts
export interface WalrusClient {
  store(data: unknown): Promise<{ blobId: string }>;
  verify(blobId: string): Promise<boolean>;
}

function hash(input: string): string {
  let h = 0;
  for (let i = 0; i < input.length; i++) h = (Math.imul(31, h) + input.charCodeAt(i)) | 0;
  return "0x" + (h >>> 0).toString(16).padStart(8, "0");
}

export class MockWalrus implements WalrusClient {
  private issued = new Set<string>();
  async store(data: unknown) {
    const blobId = hash(JSON.stringify(data) + this.issued.size);
    this.issued.add(blobId);
    return { blobId };
  }
  async verify(blobId: string) {
    return this.issued.has(blobId);
  }
}
```

```ts
import type { AgentId, Memory, NamespaceId, Policy } from "./types";
import { MockWalrus, type WalrusClient } from "./walrus";

const seed = (): { memories: Memory[]; policy: Policy } => ({
  memories: [
    { memoryId: "m1", namespace: "diet", content: "Prefers vegan meals", sourceAgent: "agent-a", walrusRef: "0x9f2ca7e1", createdAt: new Date(0).toISOString() },
    { memoryId: "m2", namespace: "health", content: "Allergic to penicillin", sourceAgent: "agent-a", walrusRef: "0xa11e0c3d", createdAt: new Date(0).toISOString() },
    { memoryId: "m3", namespace: "project", content: "Building Carry", sourceAgent: "agent-a", walrusRef: "0x41b80c3d", createdAt: new Date(0).toISOString() },
  ],
  policy: {
    "agent-a": { diet: true, health: true, project: true, billing: false },
    "agent-b": { diet: true, health: true, project: true, billing: false },
  },
});

class Store {
  private state = seed();
  private seq = 100;
  readonly walrus: WalrusClient = new MockWalrus();

  list() { return this.state.memories; }
  getPolicy() { return this.state.policy; }
  setAccess(agent: AgentId, ns: NamespaceId, allowed: boolean) { this.state.policy[agent][ns] = allowed; }
  reset() { this.state = seed(); this.seq = 100; }

  add(input: { namespace: NamespaceId; content: string; sourceAgent: AgentId }): Memory {
    const memoryId = "m" + ++this.seq;
    const ref = "0x" + Math.abs(this.hash(input.content)).toString(16).slice(0, 8).padStart(8, "0");
    const m: Memory = { memoryId, walrusRef: ref, createdAt: new Date().toISOString(), ...input };
    this.state.memories.push(m);
    return m;
  }
  private hash(s: string) { let h = 0; for (let i = 0; i < s.length; i++) h = (Math.imul(31, h) + s.charCodeAt(i)) | 0; return h; }
}

const g = globalThis as unknown as { __carryStore?: Store };
export const store = g.__carryStore ?? (g.__carryStore = new Store());
```

- [ ] **Step 1: Implement `/api/chat` (the gate path)**

Create `frontend/app/api/chat/route.ts`:
```ts
import { store } from "@/lib/store";
import { getLLM } from "@/lib/adapters";
import { recall } from "@/lib/gate";
import { buildReceipt } from "@/lib/receipt";

export async function POST(req: Request) {
  const { agentId, query } = await req.json();
  const { memories, blockedNamespaces } = recall(agentId, query, store.list(), store.getPolicy());
  const answer = await getLLM(agentId).complete({ agentId, query, memories });
  const verifiedRefs = new Set(memories.map((m) => m.walrusRef));
  const receipt = buildReceipt({
    agentId,
    answerId: "ans-" + store.list().length + "-" + memories.length,
    used: memories,
    blockedNamespaces,
    verifiedRefs,
    createdAt: new Date().toISOString(),
  });
  return Response.json({ answer, receipt });
}
```

```ts
import type { AgentId, AnswerReceipt, Memory, NamespaceId, Policy } from "./types";

const post = (url: string, body: unknown) =>
  fetch(url, { method: "POST", headers: { "content-type": "application/json" }, body: JSON.stringify(body) }).then((r) => r.json());

export const getMemories = (): Promise<{ memories: Memory[] }> => fetch("/api/memories").then((r) => r.json());
export const capture = (m: { namespace: NamespaceId; content: string; sourceAgent: AgentId }) => post("/api/memories", m);
export const sendChat = (agentId: AgentId, query: string): Promise<{ answer: string; receipt: AnswerReceipt }> => post("/api/chat", { agentId, query });
export const getPolicy = (): Promise<{ policy: Policy }> => fetch("/api/policy").then((r) => r.json());
export const setAccess = (agentId: AgentId, namespace: NamespaceId, allowed: boolean): Promise<{ policy: Policy }> => post("/api/policy", { agentId, namespace, allowed });
export const anchorReceipt = (receipt: AnswerReceipt): Promise<{ blobId: string; verified: boolean }> => post("/api/anchor", { receipt });
export const resetDemo = () => post("/api/reset", {});
```
